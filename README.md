# Анализ внутриигровых покупок «Секреты Темнолесья»

## Задача
Изучить влияние характеристик игроков и их персонажей на покупку внутриигровой
валюты «райские лепестки», а также оценить активность игроков при совершении
внутриигровых покупок.

## Данные
7 таблиц PostgreSQL (схема fantasy): игроки, покупки, расы, классы, умения,
предметы, страны. 22 214 зарегистрированных игроков, 1 307 678 транзакций.

## Что сделала

**Часть 1 — Исследовательский анализ данных**
- Доля платящих игроков по всем данным и в разрезе расы персонажа
- Статистика по стоимости покупок: min, max, среднее, медиана, стандартное отклонение
- Выявление аномальных покупок с нулевой стоимостью (907 штук, ~0,07%)
- Сравнение активности платящих и неплатящих игроков
- Рейтинг популярности эпических предметов

**Часть 2 — Ad hoc анализ**
- Зависимость покупательской активности от расы персонажа
- Сравнение количества и стоимости покупок по 7 расам

## Ключевые результаты

| Показатель | Значение |
|-----------|---------|
| Доля платящих игроков | ~18% (3 929 из 22 214) |
| Всего покупок | 1 307 678 |
| Медиана стоимости покупки | 74,86 лепестков |
| Среднее значение | 525,69 лепестков |
| Самый популярный предмет | Book of Legends (~77% всех продаж) |

Разрыв между средним и медианой указывает на редкие крупные покупки, смещающие среднее вверх.

**Раса и активность:** Human — наибольшее количество покупок (121 на игрока),
Northman — наибольшая средняя стоимость одной покупки (761 лепесток).
Различия между расами есть, но не критичны.

## Инструменты
SQL (PostgreSQL) — агрегации, CTE, PERCENTILE_DISC, STDDEV, оконные функции
